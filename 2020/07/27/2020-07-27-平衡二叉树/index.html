平衡二叉树

2020-07-27 • 分类: 数据结构

1、平衡二叉树的几个相关定义

  • 平衡因子

    BF(T) = hL = hR

    平衡因子 = 左子树高度 - 右子树高度

  • 查找长度ASL

    查找算法的查找成功时的平均查找长度的缩写，是为确定记录在查找表中的位置，需和给定值进行比较的关键字个数的期望值。

2、平衡二叉树（AVL树）的定义

  • 平衡二叉树任意结点左右子树高度之差不大于1，即 |BF(T)| <= 1；

  • 平衡二叉树可以为空；

  • n个结点的平衡二叉树的最大高度为O(log2n)；

    证明：

    • 设 n(h) 为高度为 h 的平衡二叉树所需要的最小结点数
    • n(h) = n(h-1) + n(h-2)
    • 可以发现与斐波那契数列具有相似之处
    • 找出规律后根据斐波那契数列求和公式得出：h = O(log2n)

    该定理同时说明，一颗AVL树的平均查找长度为O(log2n)

3、平衡二叉树的调整

在为AVL树插入新结点时，有时二叉树的平衡会被破坏（即二叉树的左右子树高度差绝对值大于1）。此时需要对树进行调整，使得树既满足查找条件，又满足平衡条件。

3.1 调整的方法

在一个AVL树中，标记每个结点的BF值。

将元素插入该树，并重新计算插入后树中每个结点的BF值。

若某一结点的BF值的绝对值大于2，则说明树的平衡被破坏。该结点被称为不平衡的“发现者”。插入结点被称为“麻烦结点”

3.2 RR旋转（右单旋）

若麻烦结点在发现者右子树的右边，则该次插入被称为RR插入，该次旋转被称为RR旋转。

旋转方法：将 BF 值为 -1 的结点移至 BF 值为 -2 的结点处，BF = -2 处结点变为 BF = -1 结点的左子树

3.3 LL旋转（左单旋）

若麻烦结点在发现者左子树的左边，则该次插入被称为LL插入，该次旋转被称为LL旋转。

旋转方法：同RR旋转

3.4 LR旋转

若麻烦结点在发现者左子树的右边，则该次插入被称为LR插入，该次旋转被称为LR旋转。

旋转方法：先进行一次LL旋转，再进行一次RR旋转

3.5 RL旋转

若麻烦结点在发现者右子树的左边，则该次插入被称为RL插入，该次旋转被称为RL旋转。

旋转方法：先进行一次RR旋转，再进行一次LL旋转
