    1、ORM

    ORM，对象关系映射（Object Relational Mapping），是指通过使用描述对象和数据库之间映射的元数据，将面向对象语言程序中的对象自动持久化到关系数据库中。
    简单来说，ORM就是通过对对象进行操作以操作数据库。

    2、元类

    什么是元类？

    在Python中，一切都是对象。而拥有创建实例对象能力的类，同样也是对象。我们可以将它赋值给一个变量，可以拷贝它，可以为它增加属性，可以将它作为函数参数进行传递。

    而元类就是指用来创建类的类。在 Python 中，元类为 type。
