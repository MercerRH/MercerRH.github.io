1、python套接字

在 python 中内置了 socket 模块用于创建套接字。

  • AddressFamily 参数：

    名称 目的
    AF_INET IPv4网络通信
    AF_INET6 IPv6网络通信
    AF_PACKET 链路层通信
    AF_UNIX, AF_LOCAL 本地通信

  • Type 参数：

    套接字类型（type） 说明
    SOKC_STREAM 字节流套接字
    SOCK_DGRAM 数据报套接字
    SOCK_SEQPACKET 有序分组套接字
    SOCK_RAW 原始套接字

2、UDP

  • upd通信过程：

    • 创建套接字
    • 发送数据
    • 关闭套接字

  • upd程序发送数据时，发送方的端口号未绑定的话，会使用系统随机分配的端口号

  • 若绑定了的话，则系统根据这个端口号来判别收到的数据是否属于该程序

1. socket() 函数创建UDP套接字

2. sendto() 函数发送数据

3. recvfrom() 函数接收数据

4. bind() 函数绑定ip及端口

3、TCP

  • TCP客户端通信过程：
    • 创建连接
    • 数据传输
    • 终止连接
  • TCP服务器通信过程
    • 创建套接字
    • 绑定ip与端口
    • listen将套接字转为被动连接
    • recv/send 接收、发送数据
  • 注意事项：
    • 客户端套接字关闭后，服务器 recv 会解堵塞，客户端返回的数据长度为0，因此可以以此判断客户端是否下线
    • 关闭 listen 后的套接字意味着被动套接字关闭了，会导致新的客户端不能够链接服务器，但是之前已经链接成功的客户端正常通信
    • 关闭 accept 返回的套接字意味着这个客户端已经服务完毕

5. bind() 函数绑定端口
