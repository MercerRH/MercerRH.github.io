1、栈

栈的操作集：

  • 创建空栈
  • 判断栈是否为空
  • 将元素压栈
  • 将元素出栈
  • 获取栈顶元素（不删除元素）

2、队列

队列的操作集：

  • 创建空队列
  • 判断队列是否为空
  • 将元素入队
  • 将元素出队
  • 查看队列头元素

队列的实现思路：

  • 链式存储
    • 队列的链式储存实现只需要使用一个指针指向链表头即可
    • 入队时将新元素插入队列尾；出队时将新元素从链表头中移除
  • 顺序存储
    • 若在尾端入队，则需在首端出队。则出队时每个元素都需要前移，造成O(n)操作
    • 若元素不前移而是记住新队头位置，则随着不断的出队入队，队头会慢慢移向队尾，造成曾经是队头的空间的浪费
    • 因此顺序存储的队列采用循环顺序表的方式存储
